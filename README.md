# 桌面整理大师 (Desktop Tidy Master)

搞怪向桌面整理软件。点击按钮后假装“整理”桌面，进度条走完后把所有图标藏到窗口下面。

## 功能

- **彩虹标题** — IDLE 状态标题逐字彩虹渐变，从赤到紫
- **进度条动画** — 4 段文案轮播 + 随机时长进度条（4-7 秒），制造“正在工作”的假象
- **图标隐藏** — 进度完成后，所有桌面图标被整齐排列到窗口下方隐藏
- **置顶 & 反最小化** — 完成后窗口始终置顶，最小化自动弹回
- **拖动跟随** — 拖动窗口后，图标自动重新排列到窗口下方
- **中英文切换** — 菜单栏实时切换界面语言
- **自动排列检测** — 启动时检测桌面是否开启了“自动排列图标”，若开启则弹窗提示

## 使用方法

```bash
python desktop_tidy.py
```

## 运行环境

- Windows 10 / Windows 11
- Python 3.10+

## 依赖

零第三方依赖，全部使用 Python 标准库：

| 模块 | 用途 |
|------|------|
| `tkinter` / `tkinter.ttk` | GUI 界面 |
| `ctypes` | Windows API 调用 |
| `random` | 进度条随机时长 |
| `math` | 图标网格排列计算 |
| `locale` | 系统语言检测 |
| `winreg` | 注册表读取（自动排列检测） |

## 警告

**本软件会移动桌面图标且不恢复！** 关闭窗口前请确保已手动将图标拖回原位，或清楚自己的图标位置。程序不会在退出时还原图标。
